# Getting started

## Prerequisites

1. Clone the repository: `git clone https://github.com/bryanwyk/bryanwongyk.com.git`

2. Install the dependencies: `yarn install`

## Development

-   To run the localhost development server: `yarn start`

## Production

`yarn build`
